# Youtube Transcriber
Uma extensão para o chrome que transcreve vídeos dos youtube em PT-BR.

## API
Nesse projeto eu criei uma API usando Flask para receber os URLs dos vídeos e transcrever ele, você pode acessar o código nesse link: https://github.com/maalpi/Audio-Transcription
<br/>
<br/>
<strong>OBS:</strong> Eu separei em dois repositorios porque inicialmente eu não tinha a intenção de fazer uma extensão, a API no Flask ainda tem como principal objetivo ser usado em um outro projeto o qual estou desenvolvendo.

## Tecnologias
+ React + Vite
+ Styled Components
+ Chrome Extensions
